# توثيق مشروع SkillBridge 🚀
## منصة التمكين المهني المدعومة بالذكاء الاصطناعي

---

### 1. نظرة عامة (Overview)
**SkillBridge** هي منصة متكاملة تهدف إلى حل مشكلة البطالة وسد الفجوة بين التعليم وسوق العمل. تستخدم المنصة أحدث تقنيات الذكاء الاصطناعي لتوفير تجربة مخصصة لكل باحث عن عمل، من تحليل المهارات وصولاً إلى التحضير للمقابلات.

### 2. الأهداف الرئيسية
*   **تمكين الشباب:** توفير الأدوات اللازمة لتحويل المهارات الأكاديمية إلى خبرات عملية.
*   **تقليل البطالة التقنية:** ربط الأشخاص بالوظائف التي تناسب قدراتهم الفعلية.
*   **التوجيه المهني المخصص:** تقديم مسارات تعلم ذكية بناءً على احتياجات السوق.

---

### 3. المميزات الأساسية (Core Features)

#### 🤖 مطابقة الوظائف الذكية (Smart Job Matching)
نظام يعتمد على **Google Gemini AI** لتحليل السير الذاتية ومطابقتها مع الفرص المتاحة. النظام لا يبحث فقط عن الكلمات المفتاحية، بل يفهم "السياق" و "الإمكانيات الكامنة" للمتقدم.

#### 🎭 مختبر المقابلات الافتراضي (Interview Lab)
*   محاكاة كاملة لمقابلة عمل (صوتية وكتابية).
*   تقييم فوري للأداء يشمل الجوانب التقنية والسلوكية.
*   أسئلة متغيرة بناءً على الوصف الوظيفي.

#### 🗺️ خارطة الطريق التعليمية (Learning Roadmap)
*   تحليل الفجوات بين مهارات المستخدم ومتطلبات الوظيفة المستهدفة.
*   اقتراح دورات تدريبية ومشاريع عملية لسد هذه الفجوات.

#### 💰 سوق الأعمال الصغيرة (Micro-Gigs)
مساحة تتيح للمستخدمين البدء في كسب الدخل وبناء معرض أعمال (Portfolio) حقيقي من خلال مشاريع قصيرة المدى أثناء مرحلة البحث عن وظيفة ثابتة.

#### 💡 مدرب الذكاء الاصطناعي العائم (Floating AI Coach)
مساعد ذكي متوفر في جميع صفحات المنصة للإجابة على أي استفسارات مهنية بشكل فوري.

### 4. البنية التكنولوجية (Tech Stack)

| المكون | التقنية المستخدمة |
| :--- | :--- |
| **الإطار البرمجي (Framework)** | Next.js 16 (App Router) |
| **لغة البرمجة** | TypeScript |
| **الذكاء الاصطناعي** | Google Gemini 1.5 Flash API |
| **قواعد البيانات** | PostgreSQL مع Prisma ORM |
| **التصميم (Styling)** | Tailwind CSS 4 & Framer Motion |
| **الهوية البصرية** | Vanilla CSS (Premium Dark Theme) |
| **إدارة النصوص (i18n)** | Next-intl (دعم كامل للعربية والإنجليزية) |

---

### 5. دليل التشغيل (Setup Guide)

1.  **المتطلبات:** Node.js v20+, PostgreSQL.
2.  **التثبيت:**
    ```bash
    npm install
    ```
3.  **إعداد البيئة:** إنشاء ملف `.env` وإضافة:
    *   `DATABASE_URL`
    *   `GEMINI_API_KEY`
    *   `NEXTAUTH_SECRET`
4.  **تشغيل التطبيق:**
    ```bash
    npm run dev
    ```

---

### 6. هيكلية المشروع (Project Structure)
*   `src/app/[locale]`: تحتوي على الصفحات الرئيسية مع دعم تعدد اللغات.
*   `src/components`: المكونات القابلة لإعادة الاستخدام (Seeker, Employer, Shared).
*   `src/lib`: المكتبات المشتركة (اتصال قاعدة البيانات، إعدادات Gemini).
*   `prisma/schema.prisma`: تعريف جداول قاعدة البيانات والعلاقات بينها.
